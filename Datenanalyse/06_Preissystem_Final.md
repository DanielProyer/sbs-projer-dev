## ✅ Geklärte Fragen

### 1. Pikett-Preis (160 CHF)

**Frage**: Warum 160 CHF statt 80 CHF?

**Antwort**:
- Pikett gilt für **ganzes Wochenende** (Samstag + Sonntag) = **2 Tage**
- 80 CHF × 2 Tage = **160 CHF (exkl. MWST)**
- Mit MWST: 160 × 1.081 = **172.96 CHF (inkl. MWST)**

**Berechnung in Monatsrechnungen:**
```
01.12.2025  KW 49  Pikett  Feiertage: 0
  Basis (2 Tage):        160.00 CHF (exkl.)
  Feiertage (0):           0.00 CHF
  ────────────────────────────────────
  Total exkl. MWST:      160.00 CHF

22.12.2025  KW 52  Pikett  Feiertage: 2
  Basis (2 Tage):        160.00 CHF (exkl.)
  Feiertage (2×80):      160.00 CHF (exkl.)
  ────────────────────────────────────
  Total exkl. MWST:      320.00 CHF
```

→ **Vollständig erklärt!** ✅

### 2. Störungs-Preise: Bergkunden vs. Normalkunden

**Frage**: Warum unterschiedliche Preise?

**Antwort**:
- **Normalkunden**: Standard-Tarife (z.B. Störung 3: 90 CHF)
- **Bergkunden**: Erhöhte Tarife (z.B. Störung 3: 165 CHF)
- Grund: Schwere Erreichbarkeit, längere Anfahrt, höherer Aufwand

**Preistabelle (exkl. MWST):**

| Störung | Bereich | Normalkunde | Bergkunde | Differenz |
|---------|---------|-------------|-----------|-----------|
| **Störung 1** | Zapfhahn | 55 CHF | 130 CHF | +136% |
| **Störung 2** | Leitungen | 55 CHF | 130 CHF | +136% |
| **Störung 3** | Kühlsystem | 90 CHF | 165 CHF | +83% |
| **Störung 4** | Fass | 45 CHF | 120 CHF | +167% |
| **Störung 5** | CO2/Gas | 45 CHF | 120 CHF | +167% |

**Zusätzlich (gilt für Normal- und Bergkunden):**
- Anfahrt <80km: +60 CHF Pauschale
- Anfahrt ≥80km: +0.72 CHF/km

### 3. Ersatzteile

**Frage**: Werden Ersatzteile extra berechnet?

**Antwort**:
- **NEIN!** Ersatzteile sind für den Kunden **immer kostenlos**
- Ersatzteilkosten trägt Heineken (oder werden intern verrechnet)
- Nur Arbeitszeit/Aufwand wird berechnet

**Konsequenz für Störungs-Preisspanne (150-250 CHF bei Störung 3):**

Die Preisspanne erklärt sich durch:
1. **Basis**: 90 CHF (Störung 3)
2. **Anfahrt**: +60 CHF (Normalkunde <80km)
3. **Komplexität/Aufwand**: Variable Komponente (0-100 CHF)
   - Einfache Störung (z.B. Thermostat-Reset): 0 CHF Extra
   - Mittlere Störung (z.B. Pumpe ersetzen): +30-50 CHF
   - Komplexe Störung (z.B. Kühlaggregat reparieren): +80-100 CHF

### 4. MWST-Ausweis in Monatsrechnungen

**Frage**: Sind Detailpreise inkl. oder exkl. MWST?

**Antwort**:
- **Detailpreise**: Alle Positionen sind **exkl. MWST**
- **Gesamtpreis**: MWST wird **nur einmal auf Gesamtsumme** berechnet
- **MWST-Satz**: 8.1%

## 🎯 Berechnungslogik für die App

### Pseudocode: Service-Preis

```python
def berechne_service_preis(anlage):
    """
    Berechnet Service-Preis für eine Anlage

    Args:
        anlage: Anlage-Objekt mit Eigenschaften

    Returns:
        Dict mit Preisaufschlüsselung
    """
    # Basis-Preis
    if anlage.nur_heineken:
        preis = 69  # Grundtarif eigen
    elif anlage.orion_tanks:
        preis = 92  # Grundtarif Orion
    else:
        preis = 92  # Grundtarif fremd

    # Zusatzleitungen
    zusatz_eigen = anlage.anzahl_leitungen_heineken - 1  # Erste ist im Grundtarif
    zusatz_fremd = anlage.anzahl_leitungen_fremd

    if zusatz_eigen > 0:
        preis += zusatz_eigen * 18
    if zusatz_fremd > 0:
        preis += zusatz_fremd * 23

    # Zweiter Standort
    if anlage.zweiter_standort:
        preis += 30

    # Bergkunde
    if anlage.betrieb.bergkunde:
        preis += 180

    # MWST
    mwst = round(preis * 0.081, 2)
    total = preis + mwst

    return {
        'basis': preis,
        'mwst': mwst,
        'total': total,
        'details': {
            'grundtarif': 69 if anlage.nur_heineken else 92,
            'zusatzleitungen_eigen': zusatz_eigen * 18,
            'zusatzleitungen_fremd': zusatz_fremd * 23,
            'zweiter_standort': 30 if anlage.zweiter_standort else 0,
            'bergpauschale': 180 if anlage.betrieb.bergkunde else 0
        }
    }
```

### Pseudocode: Störungs-Preis

```python
def berechne_stoerung_preis(bereich, betrieb, anfahrt_km, wochenende, komplexitaet):
    """
    Berechnet Störungs-Preis

    Args:
        bereich: 1-5 (Störungsbereich)
        betrieb: Betrieb-Objekt
        anfahrt_km: Anfahrtsweg in km
        wochenende: Boolean
        komplexitaet: 'einfach', 'mittel', 'komplex'

    Returns:
        Dict mit Preisaufschlüsselung
    """
    # Basis-Preise
    preise_normal = {1: 55, 2: 55, 3: 90, 4: 45, 5: 45}
    preise_bergkunde = {1: 130, 2: 130, 3: 165, 4: 120, 5: 120}

    # Basis (Normal- oder Bergkundepreis)
    if betrieb.bergkunde:
        basis = preise_bergkunde[bereich]
    else:
        basis = preise_normal[bereich]

    # Anfahrt (gilt für Normal- und Bergkunden gleichermassen)
    if anfahrt_km < 80:
        anfahrt = 60
    else:
        anfahrt = round(anfahrt_km * 0.72, 2)

    # Wochenend-Zuschlag
    wochenend_zuschlag = 100 if wochenende else 0

    # Komplexität
    komplexitaet_map = {
        'einfach': 0,
        'mittel': 50,
        'komplex': 100
    }
    komplexitaet_zuschlag = komplexitaet_map.get(komplexitaet, 0)

    # Zwischensumme
    preis = basis + anfahrt + wochenend_zuschlag + komplexitaet_zuschlag

    # MWST
    mwst = round(preis * 0.081, 2)
    total = preis + mwst

    return {
        'basis': preis,
        'mwst': mwst,
        'total': total,
        'details': {
            'stoerung_basis': basis,
            'anfahrt': anfahrt,
            'wochenende': wochenend_zuschlag,
            'komplexitaet': komplexitaet_zuschlag
        }
    }
```

### Pseudocode: Pikett-Preis

```python
def berechne_pikett_preis(anzahl_feiertage):
    """
    Berechnet Pikett-Preis für ein Wochenende

    Args:
        anzahl_feiertage: 0, 1, 2, ...

    Returns:
        Dict mit Preisaufschlüsselung
    """
    # Basis: Wochenende (Sa + So)
    basis = 160  # 80 × 2 Tage

    # Feiertage
    feiertage = anzahl_feiertage * 80

    # Zwischensumme
    preis = basis + feiertage

    # MWST
    mwst = round(preis * 0.081, 2)
    total = preis + mwst

    return {
        'basis': preis,
        'mwst': mwst,
        'total': total,
        'details': {
            'wochenende': basis,
            'feiertage': feiertage,
            'anzahl_feiertage': anzahl_feiertage
        }
    }
```
