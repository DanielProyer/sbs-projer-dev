## Technische Anforderungen für die App

### Automatische Rechnungserstellung

**Datenquellen:**
1. Störungen aus "Störung"-Blatt (Excel)
2. Eigenaufträge aus "Eigenauftrag"-Blatt
3. EE_Reinigungen aus "EE_Reinigung"-Blatt
4. Montagen aus "Montage"-Blatt
5. Pikett aus "Pikett"-Blatt
6. Berghäuser aus Anlagen-Stammdaten (Flag "Bergkunde")
7. Gratisreinigungen aus "Reinigung"-Blatt (spezieller Flag)

**Filterlogik:**
```
Monat = Oktober 2025
  → Filtere alle Aufträge mit Datum zwischen 01.10.2025 und 31.10.2025
  → Gruppiere nach Kategorie
  → Summiere pro Kategorie
  → Erstelle Rechnungspositionen
  → Berechne MWST (8.1%)
  → Generiere PDF
```

### Berechnungslogiken

#### Störungen
```python
# Bereich-Code bestimmt Preis
if bereich == "3":
    betrag = 150.00 or 250.00  # je nach Störungsart
elif bereich == "5":
    betrag = 105.00
elif bereich == "1+5":
    betrag = 160.00 or 310.00
# etc.
```

#### Montagen
```python
# Stunden × Stundensatz
betrag = stunden × 80.00
# Beispiel: 3.45h × 80 = 276.00 CHF
```

#### Pikett
```python
# Basis + Feiertagszuschlag
betrag = 160.00 + (anzahl_feiertage × 80.00)
```

#### Berghäuser
```python
# Wenn Anlage.Bergkunde == True:
betrag = 180.00
```

## Workflow: Monatliche Heineken-Rechnung

```

  2. Filtere nach Heineken-relevanten Aufträgen
  3. Gruppiere nach Kategorien
```

### 2. Überprüfung (manuell)
```
Benutzer überprüft:
  - Sind alle Störungen erfasst?
  - Sind alle Montagen korrekt?
  - Sind Berghäuser richtig markiert?
  - Stimmen die Beträge?
```

### 3. Generierung (automatisch)
```
App generiert:
  - Übersicht mit Summen
  - Detaillierte Positionslisten
  - PDF mit Heineken-Layout

```

```
Benutzer:
  - Sichtet PDF
  - Versendet per Email an kreditoren.ch@heineken.com
  - Markiert als "Versendet"
```

### 5. Zahlungsüberwachung (automatisch)
```
App überwacht:
```
